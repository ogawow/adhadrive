# アドハドライブ (AdhaDrive)

ADHDの特性を活かした革新的なタスク管理アプリ

## 概要

アドハドライブは、ADHDを持つ人が「5秒で忘れる」という制約を克服し、先輩からの指示を確実に記録できるアプリです。**恐怖の代替刺激**と**信頼ポイントシステム**により、継続的なタスク管理を実現します。

## 特徴

### 核心機能
- **5秒で忘れる制約に対応**: 1タップでの瞬間記録
- **恐怖の代替刺激**: 赤い画面、点滅、振動による緊急性の演出
- **信頼ポイントシステム**: 依頼者別の信頼度を数値化・可視化
- **動的緊急度表示**: 期限に応じてアイコンサイズがリアルタイム変化

### 新機能（v2.0）
- **依頼者管理システム**: 複数の依頼者を個別管理
- **信頼ポイント計算**: 速度・優先度・期限を考慮した動的計算
- **軌道アニメーション**: ポイント獲得時の美しい視覚効果
- **依頼者詳細ページ**: 統計情報と関連タスクの表示

## 技術スタック

### フロントエンド
- **Pure HTML/CSS/JavaScript**: 依存関係なしの軽量実装
- **Web標準API**: MediaRecorder API, Vibration API
- **LocalStorage**: ローカルデータ永続化
- **CSS Animations**: 滑らかなアニメーション

### 音声機能
- **Web MediaRecorder API**: ブラウザ標準の音声録音
- **Vibration API**: 触覚フィードバック

## 使用方法

### 1. 依頼者管理
- **依頼者追加**: 「新しい依頼者を追加」ボタンで依頼者を登録
- **依頼者選択**: 一覧から依頼者をクリックして選択
- **詳細確認**: 「詳細」ボタンで統計情報と関連タスクを確認

### 2. 緊急モード
- **緊急モード開始**: 「緊急モード開始」ボタンをタップ
- **5秒カウントダウン**: 赤い画面が点滅し、緊急性を演出
- **1タップ録音**: 「今すぐ記録」ボタンで音声録音開始

### 3. 通常タスク追加
- **タスク作成**: 「通常タスク追加」ボタンで手動タスク作成
- **詳細設定**: タイトル、説明、優先度、期限を設定
- **依頼者選択**: どの依頼者からのタスクかを選択

### 4. 信頼ポイントシステム
- **タスク完了**: 開始・完了ボタンで進捗管理
- **ポイント計算**: 速度・優先度・期限を考慮した自動計算
- **軌道アニメーション**: ポイント獲得時の美しい視覚効果
- **レベル更新**: 信頼レベルが自動で更新

## 主要機能

### 緊急モード機能
- **視覚的刺激**: 赤い画面と点滅アニメーション
- **時間的プレッシャー**: 5秒カウントダウン
- **振動フィードバック**: 緊急時の振動
- **1タップ操作**: 超シンプルな操作

### 信頼ポイントシステム
- **動的計算**: 基本ポイント + 優先度ボーナス + 速度ボーナス + 期限ボーナス
- **信頼レベル**: 低（0-99）・中（100-199）・高（200+）の3段階
- **視覚的表示**: アイコンの右上にポイント数表示
- **軌道アニメーション**: ポイント獲得時の美しいアニメーション

### 依頼者管理システム
- **カード表示**: 各依頼者を美しいカードで表示
- **統計情報**: 信頼ポイント、完了タスク数、信頼レベル
- **詳細ページ**: 基本情報、統計ダッシュボード、関連タスク
- **選択機能**: クリックで依頼者を選択可能

### 動的緊急度表示
- **リアルタイム更新**: 30秒間隔でアイコンサイズを更新
- **緊急度計算**: 期限と優先度を組み合わせた計算
- **サイズ変化**: 期限超過（最大）→ 6時間以内（大）→ 1日以内（中）→ 3日以内（小）
- **色分け**: 緊急度に応じた色とアニメーション

## アーキテクチャ

### ファイル構成
```
AdhaDrive/
├── web-app/
│   └── index.html          # メインWebアプリ
├── presentation/
│   ├── adhadrive_presentation.html
│   └── adhadrive_presentation.md
```

### データフロー
1. **依頼者選択** → 現在の依頼者を設定
2. **タスク作成** → 緊急モードまたは通常タスク追加
3. **タスク実行** → 開始・完了で時刻記録
4. **ポイント計算** → 速度・優先度・期限を考慮
5. **信頼更新** → 依頼者の信頼ポイントに追加
6. **視覚化** → 軌道アニメーションで表示
